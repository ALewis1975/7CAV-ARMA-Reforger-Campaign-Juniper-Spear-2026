# 7CAV ARMA Reforger Campaign Juniper Spear 2026

A modern adaptation of 1st Battalion, 7th Cavalry's "Campaign Juniper Spear" rebuilt for ARMA Reforger and a modern RHS-based combined-arms package.

## Current direction

This repository is no longer trying to force a literal Mogadishu conversion.

The campaign is now being reframed as a Reforger-first operation that preserves the original Juniper Spear arc while shifting the setting to a fictional Horn-of-Africa-inspired theater built around terrains that support:

- large-scale mounted maneuver
- helicopter insertions and casualty evacuation
- coastal lodgment and airfield seizure
- dense urban objectives
- inland pursuit and border interdiction
